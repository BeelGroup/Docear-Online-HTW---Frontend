    Docear

    Docear (dog-ear) is an academic literature suite. It integrates everything you need to search, organize and create academic literature into a single application: digital library with support for pdf documents, reference manager, note taking and with mind maps taking a central role. What's more, Docear works seamlessly with many existing tools like Mendeley, Microsoft Word, and Foxit Reader. Docear is free and open source, based on Freeplane, funded by the German Federal Ministry of Technology and developed by scientists from around the world, among others from OvGU, and the University of California, Berkeley.

    Online Mind Mapping

    This online version of Docear is currently developed in association with the HTW-Berlin. This version is in a very early beta state.

    Usage

    Registration/Login

    To use Docear-Online you need a valid Docear account which can be created at docear.org.
    To sign in, simply use the same credentials as you use to sign in at docear.

    Loading Mind Maps

    Mind maps can only be loaded if they were created previously using the Docear Desktop Application. It can be downloaded from: the official page and you uploaded them to docear.

    Navigation

    Load a Mind Map

    To load one of your personal mind maps, select it from the top right menu "select MindMap". Depending on the size of the mind map (number of nodes) it may take a several moments to display it.

    Zoom

    You can "zoom in/out" into the mind map to resize it. It is especially useful when huge mind maps are loaded to get an overview. To use the zoom functionality, you can either use the mouse wheel or the controls on the left side of the mind map. Depending on your browser version the zoom functionality may not work properly or very slow since it is using new technology. We strongly recommend to use the latest versions of browsers. This is not only due to of missing features but also because of critical security risks.

    Fold/Unfold nodes

    To fold and unfold nodes, move your mouse over a node. A little symbol appears next to the node. Click it to hide all sub nodes. Depending on the size of your mind map, it may take several moments until it is folded. While changing the tree, connections between the nodes are correctly rendered once all nodes are realigned.

    Keyboard

    It is possible to navigate through the mind map using your keyboard. Since this version currently only supports displaying mind maps, you can select nodes using the arrow keys. Selected nodes are highlighted green. Using the - or + key of your keyboard you can fold/unfold the selected node.

    Problems

    Since this is a beta version of Docear Online, there may always be some problems that occur while using the system. Anyway, some problems can be solved by following these instructions.

    I can't select a mind map

    To load a mind map, you need to have one saved and uploaded, using the Docear Desktop Application.

    I can't edit my mind map

    This is only a first beta of Docear Online which only allows to display mind maps. We are currently developing features to edit the mind map.

    I don't see my workspace?

    Docear Online doesn't support Docears workspace feature.
